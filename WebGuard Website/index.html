ABOUT US

WebGuard is a simple yet effective website blocker application that allows users to block
access to specific websites on their computer. The application uses the hosts file on the
operating system to redirect blocked website requests to the local host IP address,
effectively blocking access to the websites. WebGuard is useful for individuals who want
to limit their access to distracting or harmful websites, as well as for organizations
that need to enforce internet usage policies.
